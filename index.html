About Me

Hi, I’m Carlos B., a graduating senior heading to De Anza College before transferring to UC Davis or UC Berkeley to major in Biology. I’ve enjoyed exploring web development this year and building projects that are both useful and fun.

As a student developer, I’ve gained confidence working with HTML, CSS, and JavaScript to build interactive, creative web pages. I’ve also learned how to debug, manage layout using Flexbox and Grid, and use GitHub to version control my work. My favorite part of this journey has been designing user-friendly interfaces that reflect my personal style while solving real problems. I’m excited to keep building and improving my skills!
